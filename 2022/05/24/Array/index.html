48. Rotate Image

rotate the image by 90 degrees (clockwise)

解法：

  1. 先将 n x n 矩阵 matrix 按照左上到右下的对角线进行镜像对称
  2. 然后再对矩阵的每一行进行反转

结果就是 matrix 顺时针旋转 90 度的结果

逆时针旋转？同理：

  1. 沿着另一条对角线翻转
  2. 再反转每一行

给你一个包含若干单词和空格的字符串 s，请你写一个算法，原地反转所有单词的顺序

解法：

  1. 先将整个字符串 s 反转
  2. 然后将每个单词分别反转
